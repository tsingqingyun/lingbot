# LIBERO Setup (Post-training and Inference)

This folder provides LIBERO evaluation scripts and the matching `libero` / `libero_train` config entries.

## 1) Environment

Base LingBot-VA dependencies:

```bash
pip install torch==2.9.0 torchvision==0.24.0 torchaudio==2.9.0 --index-url https://download.pytorch.org/whl/cu126
pip install websockets einops diffusers==0.36.0 transformers==4.55.2 accelerate msgpack opencv-python matplotlib ftfy easydict
pip install flash-attn --no-build-isolation
```

Post-training and LIBERO evaluation dependencies:

```bash
pip install lerobot==0.3.3 scipy wandb --no-deps
pip install libero-libero
```

If your environment cannot resolve `libero-libero`, install LIBERO from source:

```bash
git clone https://github.com/Lifelong-Robot-Learning/LIBERO.git
cd LIBERO
pip install -e .
```

## 2) Dataset

Recommended training dataset from upstream LingBot-VA README:

- LIBERO LONG: `https://drive.google.com/file/d/1QGNkvsb1hlRmRkKCgFlyWitv17sRuagS/view`

Set dataset path for post-training:

```bash
export LINGBOT_LIBERO_DATASET_PATH=/path/to/libero_long_lerobot
```

Dataset should follow LeRobot layout and include `latents/` generated by Wan2.2 VAE.

## 3) Model Paths

```bash
export LINGBOT_LIBERO_MODEL_PATH=/path/to/checkpoint_step_xxx
export LINGBOT_WAN_BASE_PATH=/path/to/wan22_root
```

`LINGBOT_LIBERO_MODEL_PATH` should contain `transformer/`.
`LINGBOT_WAN_BASE_PATH` should contain `vae/`, `tokenizer/`, and `text_encoder/`.

## 4) Post-training

```bash
NGPU=8 CONFIG_NAME=libero_train bash script/run_va_posttrain.sh
```

## 5) Inference / Evaluation

Start server:

```bash
bash evaluation/libero/launch_server.sh
```

Run client:

```bash
bash evaluation/libero/launch_client.sh
```

Optional variables for client split:

```bash
START=0 END=10 LIBERO_BENCHMARK=libero_10 TEST_NUM=50 PORT=29056 OUT_DIR=outputs/libero \
bash evaluation/libero/launch_client.sh
```
